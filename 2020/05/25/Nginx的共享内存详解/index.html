2020-05-25 Nginx Nginx 0 评论 字数统计: 658(字) 阅读时长: 3(分)

1. 简述

1.1 应用介绍

nginx使用共享内存的模块有ngx_http_file_cache_module、ngx_http_limit_conn_module、ngx_http_limit_req_module等模块。无一例外，这几个模块都是使用nginx实现的红黑树，基于共享内存来保存他们所需要的数据。
